# 📦 Requirements, User Stories and Test Cases – Shipping Module
---
Este repositorio presenta la documentación de requerimientos, historias de usuario y casos de prueba para el módulo de gestión de envíos de un sistema de e-commerce, desarrollado en el marco de la asignatura Ingeniería y Calidad de Software de la Universidad Tecnológica Nacional – Facultad Regional Mendoza.

## Objetivo del módulo

El Módulo de Gestión de Envíos permite administrar y realizar el seguimiento de los envíos asociados a pedidos realizados en una plataforma de comercio electrónico. Entre sus funcionalidades principales se incluyen:
- Visualización del listado de envíos
- Filtrado y búsqueda de envíos
- Ordenamiento y paginación de resultados
- Visualización del detalle del envío
- Registro de observaciones ante incidencias
- Actualización manual del estado del envío
- Integración con APIs de servicios de courier

## Alcance del análisis
El trabajo contempla el análisis completo del módulo desde la perspectiva de análisis funcional y aseguramiento de calidad, incluyendo:

- Especificación de requerimientos funcionales y no funcionales
- Definición de épicas e historias de usuario
- Criterios de aceptación
- Diseño de casos de prueba positivos y negativos
- Trazabilidad entre requerimientos, historias y testing
